Privacy Policy - NADI Bulk Upload Extension

Last updated: January 3, 2025

1. Information We Collect

The NADI Bulk Upload Extension collects minimal data necessary for functionality:

  • Authentication Data: PEDI ID, User ID, Company ID, and authentication tokens from your NADI admin session
  • Username: Your NADI system username for display purposes
  • Session Information: Temporary data to maintain your login state

2. How We Use Your Information

The collected information is used solely to:

  • Authenticate API requests to the NADI system
  • Display your current session status in the extension
  • Process bulk customer uploads to your NADI account
  • Maintain session continuity during your browser session

3. Data Storage and Security

  • Local Storage Only: All data is stored locally in your browser using Chrome's storage API
  • No External Servers: We do not send your data to any external servers or third parties
  • Temporary Storage: Authentication data is only stored temporarily for your current session
  • Automatic Cleanup: Data is automatically cleared when you close your browser or log out

4. Data Sharing

We do not share, sell, or distribute your personal information to any third parties. Your data remains:

  • Stored locally on your device only
  • Used exclusively for NADI system integration
  • Never transmitted to external services

5. CSV File Processing

  • CSV files are processed locally in your browser
  • Customer data from CSV files is sent directly to NADI's official API
  • We do not store or retain any customer information from your uploads
  • All processing happens in real-time without data retention

6. Permissions Explanation

  • activeTab: To interact with the NADI admin website
  • storage: To temporarily save your authentication session
  • webRequest: To automatically detect your NADI authentication
  • scripting: To inject the upload interface on NADI pages
  • host permissions: Limited to admin.nadi.my and api.nadi.my only

7. Data Retention

  • Authentication data: Cleared when browser is closed or extension is disabled
  • Session data: Automatically expires after inactivity
  • No permanent data storage or user tracking

8. Your Rights

You have the right to:

  • Clear all stored data by disabling the extension
  • View what data is stored using the extension's debug feature
  • Uninstall the extension at any time to remove all data

9. Third-Party Services

This extension only communicates with:

  • NADI Official API (api.nadi.my): For customer uploads and system integration
  • NADI Admin Portal (admin.nadi.my): For user interface integration
  • No other third-party services are used

10. Children's Privacy

This extension is designed for business use by authorized NADI administrators and is not intended for use by children under 13.

11. Changes to Privacy Policy

We may update this privacy policy occasionally. Users will be notified of significant changes through the extension or Chrome Web Store updates.

12. Contact Information

For privacy-related questions or concerns:

  • Technical Support: 011-2610 1206
  • Extension Issues: Report through Chrome Web Store

13. Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • Google's Privacy Requirements for Extensions
  • Malaysian Personal Data Protection Act (PDPA) principles

This privacy policy is specific to the NADI Bulk Upload Extension and does not cover the NADI system itself, which has its own privacy policies and terms of service.
